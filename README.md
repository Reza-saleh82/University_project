# English Learning Platform 🌍

یک پلتفرم آموزش زبان انگلیسی مدرن و کاملاً responsive با قابلیت تعویض تم روز و شب.

## ✨ ویژگی‌ها

- **تم روز و شب** 🌙☀️
  - تم روشن با رنگ‌های سفید و آبی
  - تم تیره با رنگ‌های مشکی و تیره
  - ذخیره ترجیحات کاربر در localStorage

- **طراحی Responsive** 📱💻🖥️
  - سازگاری کامل با گوشی‌های هوشمند
  - سازگاری با تبلت‌ها
  - بهینه‌سازی برای صفحات دسکتاپ
  - منو همبرگری برای دستگاه‌های کوچک

- **صفحات مختلف**
  - صفحه اصلی (Home)
  - صفحه درس‌ها (Lessons)
  - صفحه پادکست‌ها (Podcasts)
  - صفحه ورود (Login)
  - صفحه درباره ما (About Us)

- **بخش آپلود فایل**
  - پشتیبانی از فرمت‌های MP3, WAV, M4A
  - Drag and Drop
  - بررسی حجم فایل

## 📁 ساختار پروژه

```
University_project/
├── index.html          # صفحه اصلی
├── login.html          # صفحه ورود
├── about.html          # صفحه درباره ما
├── style.css           # تمام استایل‌ها
├── script.js           # تمام کد جاوااسکریپت
└── README.md           # این فایل
```

## 🚀 شروع به کار

1. فایل‌های را از مخزن دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید
3. از منو برای حرکت بین صفحات استفاده کنید
4. برای تعویض تم بر روی آیکن ماه یا خورشید کلیک کنید

## 🎨 رنگ‌های استفاده شده

### تم روز (Light Theme)
- پس‌زمینه: سفید (#ffffff)
- متن: مشکی (#000000)
- دکمه‌ها: آبی (#007bff)

### تم شب (Dark Theme)
- پس‌زمینه: مشکی تیره (#1a1a1a)
- متن: سفید (#ffffff)
- دکمه‌ها: آبی تیره (#0066cc)

## 📱 Breakpoints (نقاط شکست)

- **دسکتاپ**: 769px و بالاتر
- **تبلت**: 481px - 768px
- **موبایل**: 480px و کمتر

## 🔧 عملکردهای JavaScript

### تغییر تم
```javascript
toggleTheme() // تبدیل بین تم روز و شب
```

### منو موبایل
```javascript
initializeHamburger() // فعال کردن منو همبرگری
```

### آپلود فایل
```javascript
handleFileUpload(event) // بررسی و آپلود فایل
```

### ورود کاربر
```javascript
handleLogin(event) // پردازش فرم ورود
```

## 📝 توضیحات کد

### HTML
- کد ساده و خوانا برای دانشجویان مقدماتی
- استفاده از تگ‌های معنایی (`<header>`, `<section>`, `<footer>`)
- فرم‌های HTML5 استاندارد

### CSS
- استفاده از CSS Variables برای رنگ‌ها
- Flexbox و Grid برای layout
- Media Queries برای responsive design
- Transitions و Transforms برای انیمیشن‌های ملایم

### JavaScript
- کد ساده بدون فریم‌ورک
- Event Listeners و DOM manipulation
- localStorage برای ذخیره ترجیحات
- توابع کوچک و قابل فهم
